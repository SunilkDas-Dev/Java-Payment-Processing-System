# 💳 Java Payment Processing System

A simple Java console-based application that simulates a basic payment process.  
It allows users to make payments using **Cash**, **Credit Card**, or **Debit Card** — showcasing the use of **method overloading** and user input handling with the `Scanner` class.

---

## 📌 Features

- Console menu with 3 payment methods
- Input validation for card numbers
- Uses method overloading to process different types of payments
- Displays formatted payment summary for users

---

## 🧱 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Method Overloading
- Scanner class for input

---

## 📂 Files

- `Payment.java`: Contains overloaded `makePayment` methods for:
  - Cash
  - Credit Card
  - Debit Card
- `PaymentProcess.java`: Contains the main method with menu logic and input handling

---

## 🚀 How to Run

Compile and run using terminal or VS Code:

```bash
javac Payment.java PaymentProcess.java
java PaymentProcess
```
